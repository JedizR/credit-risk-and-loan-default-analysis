# Credit Risk & Loan Default Analysis Dataset
> Realistic Borrower Data for Classification and Financial Risk Modeling


### About Dataset

#### Content

This dataset contains structured financial, demographic, and behavioral information of loan applicants used for credit risk assessment and loan default prediction. It includes key borrower attributes such as income level, employment status, credit history, existing liabilities, loan amount, loan purpose, repayment tenure, and other relevant financial indicators. The dataset is designed in a tabular format suitable for machine learning classification models, enabling the prediction of whether a borrower is likely to default on a loan. Each record represents an individual applicant, with features carefully selected to reflect real-world credit evaluation factors used by financial institutions.

#### Context

The Loan Risk Prediction Dataset is developed to support research and practical implementation of credit risk modeling and financial analytics. In the modern banking and fintech industry, accurate risk assessment is essential to minimize financial losses and improve lending decisions. This dataset simulates realistic borrower profiles to help data scientists, analysts, and researchers build predictive models for loan approval and default detection. It can be used for educational purposes, machine learning projects, portfolio development, and experimentation with classification algorithms such as Logistic Regression, Random Forest, Gradient Boosting, and Neural Networks.

#### Column Description

| Column Name | Data Type | Description |
| :---- | :---- | :---- |
| Age | Numerical | Age of the loan applicant in years |
| Income | Numerical | Applicant’s income (monthly or annual as defined in dataset) |
| LoanAmount | Numerical | Total loan amount requested by the applicant |
| CreditScore | Numerical | Credit score representing the applicant’s creditworthiness |
| YearsExperience | Numerical | Total years of professional work experience |
| Gender | Categorical | Gender of the applicant |
| Education | Categorical | Highest education level completed by the applicant |
| City | Categorical | City of residence of the applicant |
| EmploymentType | Categorical | Type of employment (e.g., Salaried, Self-Employed) |
| LoanApproved | Categorical (Target Variable) | Indicates whether the loan was approved (Yes/No) |
